## 项目运行

```
git clone https://github.com/bailicangdu/vue2-manage  

cd vue2-manage  

npm install 或 yarn(推荐)

npm run dev (访问线上后台系统)

npm run local (访问本地后台系统，需运行node-elm后台系统)


访问: http://localhost:8002

```
