## 一、项目介绍

该项目是基于 Vue 和 ElementUI 组件库 搭建的一个后台管理系统，能实现管理用户账号，商品分类，商品信息，订单，通过 Echarts 实现了不同商家、不同地区的销售数据可视化

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```
